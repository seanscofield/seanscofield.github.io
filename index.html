Here are some of my side projects

DOWNLOADAL (Link) -  Google Chrome extension that finds download links to songs playing on Pandora. The extension inserts a download button into the Pandora webplayer using jQuery, which, when clicked, sends the current song information to a python server. The server then scrapes the web for download links to that song, returning them to the user.

HIPPO -  Android music application that allows a user to create a music queue on the spot that other nearby users can connect and contribute to, either by adding songs to the queue or by upvoting/downvoting them. Rather than using bluetooth to connect devices, the app uses Wifi or 4G to communicate with a node.js server that can interface with mongoDB, which is where all of the music queue information is stored. This app is currently a work in progress, and I hope to integrate Spotify's api in as a next step.

TABOOZE -  An html-based version of the popular Taboo game. Originally intended to be a drinking game that combines elements of Cards Against Humanity with Taboo, the next step in this project's development would have been to port it to iPhone and Android devices using PhoneGap.

SQUABBLE (Link) -  Web app built with two other people at Berkeley's 2014 HackJam that lets people put their arguments online for others to vote on who is right. It was built using GQL (a Google SQL-like language) and html/css/javascript in 12 hours, and won the Most Addicting Hack award.

DEEP BLUE AND GOLD (Link) - Built with two other people for my EE149 (Embedded Systems) final project. Constructed a plexiglass chess board that provides hints and legal moves for a human player while he/she plays a computer. The hints, legal moves, and computer moves are indicated by Neopixel LED strips underneath the board that allow certain squares to light up. I programmed the AI for the board, and helped wire the circuitry.
